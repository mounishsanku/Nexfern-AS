# Customer Onboarding Guide

Follow this guide to get started with Nexfern FinanceOS.

## Step 1: First Login
- Log in using your assigned credentials.
- Navigate to the dashboard.

## Step 2: Create First Entity
- Go to Settings -> Entity Settings.
- Provide the necessary legal information.

## Step 3: Setup Chart of Accounts
- Review the default chart of accounts.
- Add any custom operational accounts needed.

## Step 4: Import Customers/Vendors
- Go to Import Center.
- Download the template, fill it out, and upload it.

## Step 5: Create First Invoice
- Navigate to Invoices -> Create.
- Select a customer, add line items, and save.

## Step 6: Generate Reports
- View the Profit & Loss statement to ensure the invoice correctly updated revenue accounts.
